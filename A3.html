上确界/下确界的性质

如果实函数 `f(x), g(x)` 在 `S` 上有定义, 且 `f(x) le g(x)`, `AA x in S`, 则 `underset(x in S) text(sup)\ f(x) le underset(x in S) text(sup)\ g(x)`, `underset(x in S) text(inf)\ f(x) le underset(x in S) text(inf)\ g(x)`.

只证第一条结论, 第二条同理. 易知对 `AA x in S`, `underset(x in S) text(sup)\ g(x) ge g(x) ge f(x)`, 所以 `underset(x in S) text(sup)\ g(x)` 是 `f(x)` 的上界. 而 `underset(x in S) text(sup)\ f(x)` 是 `f(x)` 的最小上界, 所以 `underset(x in S) text(sup)\ f(x) le underset(x in S) text(sup)\ g(x)`. □

应用确界原理证明戴德金原理

设 (A, B) 为实数域的一个戴德金分划, 则由 `A != O/, B != O/` 和 "对 `AA x in A`, `AA y in B` 都成立 `x lt y`" 知 A 非空有上界 (B 中的任一数都是 A 的上界). 应用确界原理知, `EE c in RR`, 使得 (1) `x le c, AA x in A`; (2) `AA epsilon gt 0`, `EE x_epsilon in A`, `c - epsilon lt x_epsilon`. 由 (2) 又得到 `AA epsilon gt 0`, `AA y in B`, `c-epsilon lt y`. 假设 `EE y_0 in B, c gt y_0`, 则对 `epsilon_0 = c - y_0 gt 0` 有 `c - epsilon_0 = y_0`, 矛盾. 故 `c le y, AA y in B`. 联系 (1) 有 ` x le c le y, AA x in A, AA y in B`. 这就证明了戴德金原理的表述二. □

应用致密性原理

子数列

从记号上看, "选取子列" 这一操作的结果是增加了下标的重数, 如 `x_n` 的子列 `x_(n_k)`. 子数列并不改变各项在原数列中的次序, 因此这里 `n_k` 是一个严格单增的正整数列, 即 `1 le n_1 lt n_2 lt ... lt n_k`. 应用归纳法可知有 `n_k ge k`. 这个不等式在许多与子数列有关的证明中有广泛应用.

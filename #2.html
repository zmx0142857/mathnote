反三角函数

定义域, 值域: `y = { ( arcsin x, if x in [-1,1], y in [-pi/2, pi/2] ), ( arccos x, if x in [-1,1], y in [0, pi] ), ( arctan x, if x in (-oo, +oo), y in (-pi/2, pi/2) ), ( text{arccot} x, if x in (-oo, +oo), y in (0, pi) ) :}`

因为 `arcsin x`, `arccos x`, `arctan x` 都在各自的定义域上单调, 所以对所有定义域中的 `x`, `{ ( sin \ arcsin x = x; ), ( cos \ arccos x = x; ), ( tan \ arctan x = x. ) :}` 反之, 则不那么优雅: `{ ( arcsin \ sin x = (-1)^n (x - n pi), if |x - n pi| le pi/2; ), ( arccos \ cos x = { ( x - n pi, if n in text{Even}; ), ( (n+1) pi - x, if n in text{Odd}; ) :}, if |x - (n + 1/2)pi| le pi/2; ), ( arctan \ tan x = x - n pi, if |x - n pi| lt pi/2. ) :}`

相应于一些三角函数公式, 对于反三角函数有如下等式 (对所有定义域中的 x): `arcsin x + arccos x = pi/2`; `arctan x + text(arccot) x = pi/2`; `2 arctan e^x - arctan \ text(sh) \ x = pi/2`; ` arcsin x = text(sgn) x * arccos sqrt (1 - x^2) = arctan {: x/sqrt (1 - x^2) :}`; `arccos x = { ( arcsin sqrt (1 - x^2), if x ge 0; ), ( pi - arcsin sqrt (1 - x^2), if x lt 0. ) :} = { ( arctan {: sqrt(1 - x^2)/x :}, if x ge 0; ), ( pi + arctan {: sqrt(1 - x^2)/x :}, if x lt 0. ) :}` ` arctan x = text(sgn) \ x * arccos {: 1/sqrt (1 + x^2) :} = arcsin {: x/sqrt (1 + x^2) :}`; ` 2 arctan sqrt ((a-b)/(a+b)) = arccos {: b/a :}`, `|a| ge |b|`; ` arccos x = 2 arccos sqrt((1+x)/2) = 2 arcsin sqrt((1-x)/2)`.

下面三个式子常用于数学分析: `arcsin x - arcsin y = { ( arcsin(x sqrt(1-y^2) - y sqrt(1-x^2)) if xy ge 0 or x^2 + y^2 le 1; ),( pi*text(sgn)(x-y) - arcsin(x sqrt(1-y^2) - y sqrt(1-x^2)) if else. ):}` ` arccos x - arccos y = arccos(xy + sqrt((1-x^2)(1-y^2)) ) * text(sgn)(y-x)`. `arctan x - arctan y = { ( arctan {: (x-y)/(1+xy) :} if xy ge -1; ), ( arctan {: (x-y)/(1+xy) + pi * text(sgn)(x-y) if xy lt -1. :} ) :}`

考虑如下的证明思路 (以 `arcsin x + arccos x = pi/2` 的证明为例):

原式成立当且仅当 `arcsin x = pi/2 - arccos x`, 注意到上式两边之差的绝对值不超过 `2 pi`, 所以上式成立当且仅当 `sin \ arcsin x = sin (pi/2 - arccos x)`, 即 `x = cos \ arccos x`, 这显然成立.

De Moivre's Theorem

`(cos x + i \ sin x)^n = cos nx + i \ sin nx`, `AA n in NN`. 对 n 作归纳即可证明. 上式左边展开, 分别取实部和虚部就得到: 对任意正整数 n, `cos nx = ((n), (0)) cos^n x - ((n), (2)) cos^(n-2) x \ sin^2 x + cdots` `+ (-1)^(epsilon/2) ((n),(epsilon)) cos^(n-epsilon) x \ sin^epsilon x`, `sin nx = ((n), (1)) cos^(n-1)x \ sin x - ((n), (3)) cos^(n-3)x \ sin^3 x + cdots` `+ (-1)^((theta-1)/20) ((n), (theta)) cos^(n-theta)x \ sin^theta x`. 其中 `epsilon` 是不超过 `n` 的最大偶数, `theta` 是不超过 `n` 的最大奇数.

Euler's Formula

`e^(ix) = cos x + i \ sin x`, `AA x in CC`. 从而 `cos x = (e^(ix) + e^(-ix))/2`, `sin x = (e^(ix) - e^(-ix))/{2i}`. `arccos x = -i \ ln (x + i sqrt(1-x^2))`, `arcsin x = -i \ ln (sqrt(1-x^2) + ix)`, `arctan x = -i \ ln {:(1 + ix)/sqrt(1+x^2):}`.

比如, 设 `y = arccos x`, 则 `e^{iy} = cos y + i \ sin y = x + i sqrt(1-x^2)`, 于是 `y = -i ln (x + i sqrt(1-x^2))`.

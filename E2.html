方程篇2: 差分方程

已知 `a x_(n+2) + b x_(n+1) + c x_n = 0`, `n = 0, 1, 2, cdots`, 且 `x_j`, `x_k` 的值已知 (1 ≤ j < k). 求数列 `x_n` 的通项.

设方程 `a x^2 + b x + c = 0` 的两根为 `alpha, beta`. 则原方程可以写作

`x_(n+1) = (alpha + beta) x_n - alpha beta x_(n-1)`, `n = 1, 2, cdots`.
(i) 若 `alpha != beta`, 由上式得到

`x_(n+1) - beta x_n = alpha (x_n - beta x_(n-1))`,

`x_(n+1) - alpha x_n = beta (x_n - alpha x_(n-1))`.

即

`x_(n+1) - beta x_n = alpha^n (x_1 - beta x_0)`,

`x_(n+1) - alpha x_n = beta^n (x_1 - alpha x_0)`.

解得
`x_n = 1/(alpha - beta)[alpha^n(x_1 - beta x_0) - beta^n (x_1 - alpha x_0)]`.
但如果 `j,k != 0,1`, 上式就不能直接使用. 我们作进一步的处理:
` {:( x_k - beta^k x_0 , = x_k - beta x_(k-1) + beta x_(k-1) - beta^2 x_(k-2) + cdots + beta^(k-1) x_1 - beta^k x_0 ),(, = sum_(i=1)^k beta^(k-i) (x_i - beta x_(i-1)) ),(, = sum_(i=1)^k beta^(k-i) alpha^(i-1) (x_1 - beta x_0) ),(, = (x_1 - beta x_0) (alpha^k - beta^k)/(alpha - beta). ):}`
从而得到
`x_1 - beta x_0 = (alpha - beta)/(alpha^k - beta^k) (x_k - beta^k x_0)`, `k ge 1`.
上式对 k = n 也成立, 所以
`(x_n - beta^n x_0)/(alpha^n - beta^n) = (x_k - beta^k x_0)/(alpha^k - beta^k)`.
即
`x_n = (alpha^n - beta^n)/(alpha^k - beta^k) (x_k - beta^k x_0) + beta^n x_0 = (alpha^n - beta^n)/(alpha^k - beta^k) x_k + (alpha^k beta^n - alpha^n beta^k)/(alpha^k - beta^k) x_0`.
所以对任意正整数 0 < j < k ≤ n, 有
` {:( x_n ,= (alpha^n - beta^n)/(alpha^k - beta^k) x_k + (alpha^k beta^n - alpha^n beta^k)/(alpha^k - beta^k) x_0 ),( ,= (alpha^n - beta^n)/(alpha^j - beta^j) x_j + (alpha^j beta^n - alpha^n beta^j)/(alpha^j - beta^j) x_0. ):}`
消去 `x_0`, 得
` [(alpha^k - beta^k) x_n - (alpha^n - beta^n) x_k] / (alpha^k beta^n - alpha^n beta^k) = [(alpha^j - beta^j) x_n - (alpha^n - beta^n) x_j] / (alpha^j beta^n - alpha^n beta^j)`.
所以
`x_n = (alpha^j beta^n - alpha^n beta^j) / (alpha^j beta^k - alpha^k beta^j) x_k - (alpha^k beta^n - alpha^n beta^k) / (alpha^j beta^k - alpha^k beta^j) x_j`.
易知上式对 j = 0 也成立.

(ii) 若 `alpha = beta`, 我们只得到

`(x_(n+1) - alpha x_n) = alpha^n (x_1 - alpha x_0)`, `n = 0, 1, 2, cdots`.
记 `c = x_1 - alpha x_0`, 上式即
`x_(n+1) - alpha x_n = c alpha^n`.
`alpha = 0` 时, 原方程有平凡解 `x_n = 0, n ge 2`. 若 `alpha != 0`, 由上式得
`x_(n+1)/alpha^n = c + x_n/alpha^(n-1) = nc + x_1`.
于是
` x_n = alpha^(n-1) [x_1 + (n-1)c] = alpha^(n-1) (alpha x_0 +nc)`.
类似上面的处理, 有
` {:( ,x_k - alpha^k x_0 ),( = ,sum_(i=1)^k alpha^(k-i) (x_i - alpha x_(i-1)) ),( = ,sum_(i=1)^k alpha^(k-i) alpha^(i-1) (x_1 - alpha x_0) ),( = ,k alpha^(k-1) (x_1 - alpha x_0). ):}`
于是
`c = x_1 - alpha x_0 = (x_k - alpha^k x_0)/(k alpha^(k-1))`.
所以
`x_n = alpha^(n-1) [alpha x_0 + n/k (x_k - alpha^k x_0) / alpha^(k-1)] = n/k alpha^(n-k) x_k - (n-k)/k alpha^n x_0`.
所以对任意正整数 0 < j < k ≤ n, 有
`x_n = n/k alpha^(n-k) x_k - (n-k)/k alpha^n x_0 = n/j alpha^(n-j) x_j - (n-j)/j alpha^n x_0`.
消去 `alpha^n x_0`, 得
` (n alpha^(n-k) x_k - k x_n) / (n-k) = (n alpha^(n-j) x_j - j x_n) / (n-j)`.
所以
` x_n = (n-j)/(j+k) alpha^(n-k) x_k - (n-k)/(j+k) alpha^(n-j) x_j`.
易知上式对 `j = 0` 也成立. □

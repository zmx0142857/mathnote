几何篇5: 球面几何

G5.1 球面三角的余弦与正弦公式

图中 `CM`, `CN` 分别是 `CA`, `CB` 的切线. 由平面三角的余弦定理,

`MN^2 = OM^2 + ON^2 - 2*OM*ON*cos c.`

`MN^2 = CM^2 + CN^2 - 2*CM*CN*cos C.`

两式相减,

`0 = 2*OC^2 + 2*CM*CN*cos c - 2*OM*ON*cos C,`

`cos c = (OC)/(ON) * (OC)/(OM) + (CN)/(ON) * (CM)/(OM) * cos C,`

即"余弦公式":

`cos c = cos a * cos b + sin a * sin b * cos C.`

图中 `CD_|_Delta AOB`, `DE_|_OA`, `DF_|_OB`. 从而 `OA_|_Delta CED`, `OB_|_Delta CDF`. 于是

`sin a/sin A = (CF)/(CO)//(CD)/(CE) = (CE*CF)/(CO*CD),`

`sin b/sin B = (CE)/(CO)//(CD)/(CF) = (CE*CF)/(CO*CD).`

即"正弦公式":

`sin a/sin A = sin b/sin B.`

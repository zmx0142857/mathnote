几何篇6: 从欧拉线到九点圆

G6.1 证明: 三角形 ABC 的外心 O, 重心 G, 垂心 H 满足 `vec(OH) = 3 vec(OG)`.

证 为了保持 ABC 三点的对称性, 取外心 O 为原点. 不妨令 ABC 外接圆的半径 R = 1, 则 A, B, C 三点的坐标分别为 `(cos theta_i, sin theta_i), i = 1, 2, 3`.

由 G6.3.1, `vec(OG) = 1/3 (sum cos theta_i, sum sin theta_i)`. 我们记点 H' 的坐标为 `(sum cos theta_i, sum sin theta_i)`, 显然 `vec(OH') = 3vec(OG)`.

由于

`vec(AH') * vec(BC) = (cos theta_3 + cos theta_2, sin theta_3 + sin theta_2) * (cos theta_3 - cos theta_2, sin theta_3 - sin theta_2) = 0`,

同样地, 将上式中的下标 (2, 3) 换成 (3, 1), 得 `vec(BH') * vec(AC) = 0`; 换成 (1, 2), 得 `vec(CH') * vec(AB) = 0`. 但三角形的垂心是唯一的. 所以 H' 和 H 其实是同一点, 即三角形 ABC 的垂心. 证毕.

注G6.1.1 本题亦 (在三角形外心和重心的存在性已证的前提下) 给出了三角形垂心的存在性的证明, 即平面上存在一点 H 使得 `AH_|_BC`, `BH_|_CA`, 且 `CH_|_AB`.

注G6.1.2 O, G, H 所在的直线就称为欧拉线.

G6.2 设三角形 ABC 的外心为 O, 内心为 I, 其外接圆, 内切圆的半径分别为 R, r. 证明: `vec(OI^2) = R^2 - 2Rr`.

证 仍取外心 O 为坐标原点. 由 G5.3.2,

`vec(OI) = 1/(a+b+c)(a vec(OA) + b vec(OB) + c vec(OC))`,

其中 a, b, c 分别是边 BC, CA, AB 的长. 显然, `vec(OA^2) = vec(OB^2)` `= vec(OC^2) = R^2`.

而

`2 vec(OA) * vec(OB) = vec(OA)^2 + vec(OB^2) - vec(AB^2) = 2R^2 - c^2`

于是

`{: ( vec(OI)^2 ,= 1/(a+b+c)^2 (a^2 vec(OA^2) + b^2 vec(OB^2) + c^2 vec(OC^2) ) + 2ab vec(OA) * vec(OB) + 2bc vec(OB) * vec(OC) + 2ca vec(OC) * vec(OA) ), ( ,= 1/(a+b+c)^2 [R^2(a^2+b^2+c^2) + ab(2R^2 - c^2) + bc(2R^2 - a^2) + ca(2R^2 - b^2)] ), ( ,= 1/(a+b+c)^2 [R^2(a+b+c)^2 - abc(a+b+c)] ), ( ,= R^2 - (abc)/(a+b+c). ) :}`

设三角形 ABC 的面积为 S. 由 G2.2, `S = (abc)/(4R) = (a+b+c)/2 * r`, 所以 `(abc)/(a+b+c) = 2Rr`.

从而得到 `vec(OI^2) = R^2 - 2Rr`.

注G6.2 这一式子就叫做欧拉公式. 我们另有"纯几何"的证明如下:

记 d = OI. 设直线 OI 与圆 O 交于点 E, F, 则 `R^2 - d^2 = (R+d)(R-d) = EI * FI`.

设直线 AI 与圆 O 的另一交点为 M, 则 `EI * FI = AI * MI`.

过 I 作 AC 的垂线, 垂足为 D, 再延长 MO 交圆 O 于 L, 连接 LB, MB.

现在, `/_ DAI = /_ MAB = /_ L`, `/_ ADI = /_ LBM = pi/2`,

所以 `Delta MBL ~ Delta IDA`.

所以 `ID * LM = AI * MB`.

因为 ID = r, LM = 2R,

所以上式左边等于 2Rr. 要证明 `d^2 = R^2 - 2Rr`,

只需证上式右边等于 `R^2 - d^2`,

即 `AI * MI`,

而这只需证 `MB = MI`.

如图所示, `alpha = /_ IAB + /_ ABI`,

其中 `/_ IAB = /_ L = /_ DAI = gamma`,

`/_ ABI = beta`.

所以由"等边对等角"知 MB = MI. 证毕.

分析篇2: 极限杂例

A2.1 证明: `lim_(n -> oo) root(n)(n) = 1.`

证: `n gt 2` 时, 有

`1 lt root(n)(n)` `= root(n)(sqrt n * sqrt n * 1)` `lt (2 sqrt n + n - 2)/n` `lt 2/sqrt n + 1.`

由两边夹法则, 命题成立.

A2.5 证明: `lim_(n -> oo) (1 + 1/n)^n` 存在.

证: 先证 `(1 + 1/n)^n` 单调递增. 由均值不等式 (A0.3) 知:

`root(n+1)((1 + 1/n)^n) lt (n(1 + 1/n) + 1)/(n+1) = 1 + 1/(n+1).`

所以 `(1 + 1/n)^n lt (1 + 1/(n+1))^(n+1).`

再证此数列有上界. `n ge 2` 时:

`(1 + 1/n)^n = 1 + n * 1/n + (n(n-1))/(2!) * 1/n^2 + cdots + 1/n^n` `lt sum_(k=0)^n 1/(k!)` `le 2 + sum_(k=2)^n 1/(k(k-1))` `= 2 + 1 - 1/n lt 3.`

从而由单调有界原理知极限 `lim_(n -> oo) (1 + 1/n)^n` 存在. 它的极限值就是我们熟知的自然对数的底 `e.`

推论A2.5.1 由极限的四则运算得到:

`lim_(n -> oo) (1 + 1/n)^(n+1) = lim_(n -> oo) (1 + 1/(n+1))^n = e,`

`lim_(n -> oo) (1 - 1/n)^n = lim_(n -> oo) (1 - 1/n)^(n+1) = lim_(n -> oo) (1 - 1/(n+1))^n = 1/e.`

推论A2.5.2 `(1 + 1/n)^n lt e lt (1 + 1/n)^(n+1), quad` `AA n in NN.`

或等价地, `1/(n+1) lt ln(1 + 1/n) lt 1/n, quad` `AA n in NN.`

证: 不等式的前半部分由 A1.5 的证明可得. 后半部分由推论 A1.5.2, `lim_(n -> oo) (1 + 1/n)^(n+1) = e,` 因此只需证明 `(1 + 1/n)^(n+1)` 严格单调递减. 由均值不等式有:

`root(n+2)(1/( (1 + 1/n)^(n+1) )) lt 1/(n+2)((n+1)/(1 + 1/n) + 1) = (n+1)/(n+2).`

所以 `(1 + 1/n)^(n+1) gt ((n+2)/(n+1))^(n+2) = (1 + 1/(n+1))^(n+2).` 证毕.

注A2.5.2 此不等式也可由微分中值定理得到.

分析篇2: 极限杂例

证明: `lim_(n -> oo) root(n)(n) = 1`.

`n gt 2` 时, 有

` 1 lt root(n)(n)` `= root(n)(sqrt n * sqrt n * 1)` `lt (2 sqrt n + n - 2)/n` `lt 2/sqrt n + 1`.
由两边夹法则, 命题成立.

证明: `lim_(n to oo) a^n/(n!) = 0`, `AA a in R`.

取 `N_1 = [|a| + 1]`, 则 `|a|/n lt 1, AA n gt N_1`. 取实数 `r` 使 `|a|/n lt r lt 1`, 有

` |a^n/(n!)| = |a|^n/(n!) le r^(n-N_1) |a|^(N_1)/(N_1!) lt r^(n-N_1) (|a|+1)^(N_1)/(N_1!)`.
记 `b = |a| + 1`, 对 `AA epsilon gt 0`, 要使上式右边小于 `epsilon`, 只要
`(n - N_1) ln r lt ln((N_1!)/b^(N_1) epsilon)`,
即
`n gt ln((N_1!)/b^(N_1) epsilon) / ln r + N_1`.
所以只要取 `N = [ln((N_1!)/b^(N_1) epsilon) / ln r + N_1] + 1`, 当 `n gt N` 时, 就有 `| a^n/(n!) | lt epsilon`. □

证明下面的数列收敛, 且极限为 `root(m)(A)`.

`x_1 = 1`,

`x_(n+1) = 1/m [(m-1) x_n + A/x_n^(m-1)]`.

(`m in ZZ`, `m != 0`; `A gt 0`)

极限的四则运算

若 `lim_(n to oo) x_n = a`, `lim_(n to oo) y_n = b`, 则

`lim_(n to oo) (x_n +- y_n) = a +- b`;

`lim_(n to oo) (x_n y_n) = ab`;

`lim_(n to oo) (x_n)/(y_n) = a/b` (当 b ≠ 0).

  1. 对 `AA epsilon gt 0`, 由 `lim_(n to oo) x_n = a` 和 `lim_(n to oo) y_n = b` 知 `EE` `N_1`, `N_2`, 使

    `|x_n - a| lt epsilon/2, AA n gt N_1`;

    `|y_n - b| lt epsilon/2, AA n gt N_2`.

    故 `n gt max{N_1, N_2}` 时,
    ` |(x_n +- y_n) - (a +- b)| le |x_n - a| + |y_n - b| lt epsilon/2 + epsilon/2 = epsilon`.
  2. 由收敛数列的有界性知 `EE` M > 0 使
    `|y_n| lt M`, `AA n in NN`.
    对 `AA epsilon gt 0`, 由 `lim_(n to oo) x_n = a` 和 `lim_(n to oo) y_n = b` 知 `EE N_1, N_2 in NN`, 使

    `|x_n - a| lt epsilon/(2M)`;

    `|y_n - b| lt epsilon/(2(|a| + 1))`.

    (这样就避免了对 a = 0 的讨论). 因此 `n gt max{N_1, N_2}` 时,
    ` {:( |x_n y_n - ab| , = |x_n y_n - a y_n + a y_n - ab| ),(, le |x_n - a| |y_n| + |a| |y_n - b| ),(, lt epsilon/(2M) M + |a| epsilon/(2(|a| + 1)) lt epsilon. ):}`
  3. 因为 `lim_(n to oo) y_n = b != 0`, 所以由数列极限的保号性知 `EE N_1 in NN` 与 `m gt 0` 使
    `|y_n| gt m, AA n gt N_1`.
    对 `AA epsilon gt 0`, 由 `lim_(n to oo) x_n = a` 和 `lim_(n to oo) y_n = b` 知存在 `N_2, N_3 in NN` 使

    `|x_n - a| lt m epsilon/2, AA n gt N_2`;

    `|y_n - b| lt (m|b|)/(|a|+1) epsilon/2, AA n gt N_3`.

    当 `n gt max{N_1, N_2, N_3}` 时有
    ` {:( |(x_n)/(y_n) - a/b| , = |(b x_n - a y_n)/(b y_n)| ),(, = (|b x_n - ab + ab - a y_n|)/(|b| |y_n|) ),(, lt (|b| |x_n - a| + |a| |y_n - b|)/(m |b|) ),(, = 1/m |x_n - a| + |a|/(m|b|) |y_n - b| ),(, lt 1/m m epsilon/2 + |a|/(m|b|) (m|b|)/(|a|+1) epsilon/2 lt epsilon. ):}` □

证明: `lim_(n -> oo) (1 + 1/n)^n` 存在.

先证 `(1 + 1/n)^n` 单调递增. 由均值不等式 (A1.1) 知:

` root(n+1)((1 + 1/n)^n) lt (n(1 + 1/n) + 1)/(n+1) = 1 + 1/(n+1)`.
所以 `(1 + 1/n)^n lt (1 + 1/(n+1))^(n+1)`. 再证此数列有上界. `n ge 2` 时:
` (1 + 1/n)^n = 1 + n * 1/n + (n(n-1))/(2!) * 1/n^2 + cdots + 1/n^n` `lt sum_(k=0)^n 1/(k!)` `le 2 + sum_(k=2)^n 1/(k(k-1))` `= 2 + 1 - 1/n lt 3`.
从而由单调有界原理知极限 `lim_(n to oo) (1 + 1/n)^n` 存在. 它的极限值就是我们熟知的自然对数的底 `e`.

由极限的四则运算得到:

` lim_(n to oo) (1 + 1/n)^(n+1) = lim_(n to oo) (1 + 1/(n+1))^n = e`,
` lim_(n to oo) (1 - 1/n)^n = lim_(n to oo) (1 - 1/n)^(n+1) = lim_(n to oo) (1 - 1/(n+1))^n = 1/e`.

`(1 + 1/n)^n lt e lt (1 + 1/n)^(n+1)`, `AA n in NN`.
或等价地,
`1/(n+1) lt ln(1 + 1/n) lt 1/n`, `AA n in NN`.

不等式的前半部分由 A2.3 的证明可得. 后半部分由 推论A2.1, `lim_(n to oo) (1 + 1/n)^(n+1) = e`, 因此只需证明 `(1 + 1/n)^(n+1)` 严格单调递减. 由均值不等式有:

` root(n+2)(1/( (1 + 1/n)^(n+1) )) lt 1/(n+2)((n+1)/(1 + 1/n) + 1) = (n+1)/(n+2)`.
所以 ` (1 + 1/n)^(n+1) gt ((n+2)/(n+1))^(n+2) = (1 + 1/(n+1))^(n+2)`. □

此不等式也可由微分中值定理得到.

行列式的导数

设 `f_(11), f_(12), cdots, f_(1n), f_(21), cdots, f_(2n), cdots` `f_(n1), ..., f_(n n)` 是关于自变元 `x` 的 n `xx` n 个可微函数, 记 `Delta = | ( f_(11)(x), f_(12)(x), cdots, f_(1n)(x) ), ( f_(21)(x), f_(22)(x), cdots, f_(2n)(x) ), ( vdots, vdots, , vdots ), ( f_(n1)(x), f_(n2)(x), cdots, f_(n n)(x) ) |`, `Delta^((i))` 是将 `Delta` 第 i 行的函数换成相应的导函数所得到的 行列式, i = 1, 2, ..., n, 则 `d/dx Delta = sum_(i=1)^n Delta^((i))`.

n = 2 时, ` d/dx |( f_(11) \ f_(12) ), ( f_(21) \ f_(22) )| = f_(11)^' f_(22) + f_(11) f_(22)^' - f_(12)^' f_(21) - f_(12) f_(21)^' = |(f_(11)^' \ f_(12)^'), (f_(21) \ f_(22))| + |(f_(11) \ f_(12)), (f_(21)^' \ f_(22)^')|`. 命题成立, 现在设命题对 n-1 成立, 则对 n, 若记 `F_(ij)` 为相应的 代数余子式, 有 ` {:(, d/dx Delta ),( =, d/dx sum_(k=1)^n f_(1k) F_(1k) ),( =, sum_(k=1)^n f_(1k)^' F_(1k) + sum_(k=1)^n f_(1k) F_(1k)^' ),( =, Delta^((1)) + sum_(k=1)^n f_(1k) sum_(j=1)^(n-1) F_(1k)^((j)) ),( =, Delta^((1)) + sum_(j=1)^(n-1) sum_(k=1)^n f_(1k) F_(1k)^((j)) ),( =, sum_(i=1)^n Delta^((i)). ):}` □

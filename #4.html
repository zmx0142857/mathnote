恒等式

`P ^^ (Q vv R) iff P ^^ Q vv P ^^ R` `^^` 在 `vv` 上的分配律
`P vv Q ^^ R iff (P vv Q) ^^ (P vv R)` `vv` 在 `^^` 上的分配律
`not (P vv Q) iff not P ^^ not Q` 德·摩根定律
`not (P ^^ Q) iff not P vv not Q`
`(P rarr Q ) iff not P vv Q` 蕴含表达式
`(P rarr Q) iff (not Q rarr not P)` 逆反律
`(P ^^ Q rarr R) iff (P rarr (Q rarr R))` 输出律(CP规则)
`(P rarr Q) ^^ (P rarr not Q) iff not P` 归谬律
` {:( (P harr Q) ,iff (P rarr Q) ^^ (Q rarr P) ),( ,iff (not P vv Q) ^^ (not Q vv P) ),( ,iff P ^^ Q vv not P ^^ not Q ),( ,iff not (not P harr Q) ):}` 等值表达式
`P uarr Q iff not (P ^^ Q)` 与非(Sheffer)
`not P iff P uarr P`
`P vv Q iff (P uarr P) uarr (Q uarr Q)`
`P ^^ Q iff (P uarr Q) uarr (P uarr Q)`
`P darr Q iff not (P vv Q)` 或非(Peirce)
`not P iff P darr P`
`P vv Q iff (P darr Q) darr (P darr Q)`
`P ^^ Q iff (P darr P) darr (Q darr Q)`

永真蕴含式

`P rArr P vv Q` 加法式
`P ^^ Q rArr P` 简化式
`P ^^ (P rarr Q) rArr Q` 假言推理
`not Q ^^ (P rarr Q) rArr not P` 拒取式(由假言推理逆反得到)
`not P ^^ (P vv Q) rArr Q` 析取三段论
`(P rarr Q) ^^ (Q rarr R) rArr P rarr R` 前提三段论
`(P rarr Q) ^^ (R rarr S) ^^　(P vv R) rArr Q vv S` 构造性二难推理
`(P rarr Q) ^^ (R rarr S) ^^ (not Q vv not S) rArr not P vv not R` 破坏性二难推理

含量词的永真公式

`{ ( AA x P(x) rArr P(y) ), ( P(y) rArr EE x P(x) ) :}`

`{ ( AA x not P(x) iff not EE x P(x) ), ( EE x not P(x) iff not AA x P(x) ) :}`

`{ ( AA x (P(x) ^^ Q(x)) iff AA x P(x) ^^ AA x Q(x) ), ( EE x (P(x) vv Q(x)) iff EE x P(x) vv EE x Q(x) ), ( EE x (P(x) ^^ Q(x)) rArr EE x P(x) ^^ EE x Q(x) ), ( AA x P(x) vv AA x Q(x) rArr AA x (P(x) vv Q(x))) :}`

`{ ( EE x (P(x) rarr Q(x)) iff AA x P(x) rarr EE x Q(x) ), ( EE x P(x) rarr AA x Q(x) rArr AA x (P(x) rarr Q(x)) ) :}`

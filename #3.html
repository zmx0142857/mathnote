附录篇3: 一些恒等式

` x^(n+1) - y^(n+1) = (x-y)(x^n + x^(n-1)y + cdots + xy^(n-1) + y^n)`, `x, y in RR, n = 1,2,cdots.`

`max{x, y} = (x+y)/2 + |x-y|/2`, `min{x, y} = (x+y)/2 - |x-y|/2`, `x,y in RR`; 特别地, `max{x, 0} = (x + |x|)/2`, `min{x, 0} = (x - |x|)/2`, `x, y in RR.`

`(vec x * vec y)^2 + |vec x xx vec y|^2 = |vec x|^2|vec y|^2`; `(ac + bd)^2 + (ad-bc)^2 = (a^2 + b^2)(c^2 + d^2), a,b,c,d in RR`.

第二个式子可用于证明如下命题: 如果两整数 m, n 都能写成两个整数的平方和, 则 mn 也能.

拉格朗日恒等式: ...

`1/(n(n+m)) = 1/m (1/n - 1/(n+m)).`

` 1/(prod_(k=0)^n (x+k)) = 1/(n!) sum_(k=0)^n (-1)^k ((n), (k)) 1/(x+k)`; `AA n in NN`; `AA x in RR`; `x+k ne 0, k = 0, 1, 2, cdots, n`.

先计算 `1/(n(n+1)(n+2))` 这样的式子找找灵感!

使用归纳法证明. `n = 0` 时, 等号两边都等于 `1/x`, 因此等式成立. 假设命题对于自然数 `n` 成立, 考察自然数 `n+1`, 有:

` 1/(prod_(k=0)^(n+1) (x+k)) = 1/(x+n+1) * 1/(n!) sum_(k=0)^n (-1)^k ((n), (k)) 1/(x+k)`.
欲证等式成立, 只需证明上式等于
`1/((n+1)!) sum_(k=0)^(n+1) (-1)^k ((n+1), (k)) 1/(x+k)`,
即证
` (n+1)/(x+n+1) sum_(k=0)^n (-1)^k ((n), (k)) 1/(x+k) = sum_(k=0)^(n+1) (-1)^k ((n+1), (k)) 1/(x+k)`. (#3-1)
由于
`1/((x+k)(x+n+1)) = 1/(n+1-k) ( 1/(x+k) - 1/(x+n+1) )`, `k = 0, 1, 2, cdots, n.`
所以 (#3-1) 的左边等于
` {:( ,sum_(k=0)^n (-1)^k (n+1)/(n+1-k) ((n), (k)) ( 1/(x+k) - 1/(x+n+1) ) ),( =, sum_(k=0)^n (-1)^k ((n+1), (k)) ( 1/(x+k) - 1/(x+n+1) ) ),( =, sum_(k=0)^n (-1)^k ((n+1), (k)) 1/(x+k) - sum_(k=0)^n (-1)^k ((n+1), (k)) 1/(x+n+1) ),(, + (-1)^(n+1) ((n+1), (n+1)) 1/(x+n+1) - (-1)^(n+1) ((n+1), (n+1)) 1/(x+n+1) ),( =, sum_(k=0)^(n+1) (-1)^k ((n+1), (k)) 1/(x+k) - sum_(k=0)^(n+1) (-1)^k ((n+1), (k)) 1/(x+n+1) ),( =, sum_(k=0)^(n+1) (-1)^k ((n+1), (k)) 1/(x+k) - 1/(x+n+1) (1-1)^(n+1) ),( =, sum_(k=0)^(n+1) (-1)^k ((n+1), (k)) 1/(x+k). ):}`
(#3-1) 成立. 所以原式成立. 证毕.

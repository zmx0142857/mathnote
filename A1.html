分析篇1: 不等式

A1.1 基本不等式

设 `a, b in RR^+`, 则由 `(sqrt a - sqrt b)^2 = a + b - 2 sqrt(ab) ge 0` 得 `sqrt(ab) le (a+b)/2`, 其中等号成立当且仅当 `a = b`.

上式两边同乘 `sqrt(ab)`, 得 `ab le (a+b)/2 sqrt(ab)`. 所以 `(2ab)/(a+b) le sqrt(ab)`.

另一方面, 在 `2ab leq a^2+b^2` 的两边同时加上 `a^2+b^2`, 有 `(a+b)^2 le 2(a^2+b^2)`, 即 `(a+b)^2/4 le (a^2+b^2)/2`, 两边同时取算术平方根就有 `(a+b)/2 le sqrt((a^2+b^2)/2)`.

综合上述三个不等式得到

`(2ab)/(a+b) le sqrt(ab) le (a+b)/2 le sqrt((a^2+b^2)/2)`.

其中三个等号成立当且仅当 `a=b`.

这四个数分别叫做 `a` 和 `b` 的 调和平均数, 几何平均数, 算术平均数与平方平均数. 将它们分别记作 `s`, `t`, `u`, `v`, 我们有

  • `1/s = 1/2(1/a + 1/b)`,
  • `ln t = 1/2(ln a + ln b)`,
  • `u = 1/2(a+b)`,
  • `v^2 = 1/2(a^2 + b^2)`.

    • A1.2 柯西不等式

    对任意非零实数 `a_1, a_2, cdots, a_n` 与 `b_1, b_2, cdots, b_n`, 下面的不等式成立:

    ` ( sum_(k=1)^n a_k b_k )^2 le ( sum_(k=1)^n a_k^2 ) ( sum_(k=1)^n b_k^2 )`,
    其中等号成立当且仅当存在实数 `lambda`, 使得 `b_k = lambda a_k`, `k=1, 2, cdots, n`.

    令 `f(x) = sum_(k=1)^n (a_k x - b_k)^2 = (sum_(k=1)^n a_k^2) x^2 - 2(sum_(k=1)^n a_k b_k) x + sum_(k=1)^n b_k^2 ge 0`. 这是 `x` 的一元二次函数, 易知其判别式

    `Delta = 4( sum_(k=1)^n a_k b_k )^2` `- 4( sum_(k=1)^n a_k^2 )( sum_(k=1)^n b_k^2 ) le 0`,
    即要证的不等式. 其中等号成立时, `Delta = 0`, 此时必存在唯一的实数 `lambda` 使得 `f(lambda) = 0`, 即成立 `b_k = lambda a_k`, `k = 1, 2, cdots, n`. 反之若存在实数 `lambda`, 使得 `b_k = lambda a_k`, `k = 1, 2, cdots, n`, 从而 `f(lambda) = 0`, 也能推得 `Delta = 0`. `square`

    A1.3 均值不等式

    设 `a_1, a_2, cdots, a_n` 是 `n` 个正实数 `(n ge 2),` 记 `S_n = n/( sum_(k=1)^n 1/(a_k) )`, `T_n = ^n sqrt( prod_(k=1)^n a_k )`, `U_n = ( sum_(k=1)^n a_k )/n`, `V_n = sqrt( ( sum_{k=1}^n a_k^2 )/n )`, 则下面的不等式成立:

    `S_n le T_n le U_n le V_n`, `(A1-3-1)`

    三个等号成立当且仅当 `a_1 = a_2 = cdots = a_n.`

    (I) 证明 `T_n le U_n.` `n=2` 时此不等式即基本不等式. 假设 `n=m` 时不等式成立, 则因为

    `U_(m+1) = ( sum_(k=1)^(m+1) a_k )/(m+1) => (m+1) U_(m+1) = sum_(k=1)^(m+1) a_k`,
    所以
    ` {:( U_(m+1) ,= (2mU_(m+1)) / (2m) = ( (m+1) U_(m+1) + (m-1) U_(m+1) ) / (2m) ),( ,= ( sum_(k=1)^(m+1) a_k + (m-1) U_(m+1)) / (2m) ),( ,= 1/2 ( (sum_(k=1)^m a_k)/m + (a_(m+1) + (m-1) U_(m+1))/m ). ):}`
    再由归纳假设和基本不等式,
    ` U_(m+1) ge ( root(m)(prod_{k=1}^m a_k) + root(m)(a_(m+1) * U_(m+1)^(m-1)) ) /2` `ge root(2m)(prod _(k=1)^(m+1) a_k * U_(m+1)^(m-1))`.
    所以 `U_(m+1)^(2m) ge prod_(k=1)^(m+1) a_k * U_(m+1)^(m-1)`, 进而有 `U_(m+1) ge root(m+1)( prod_(k=1)^(m+1) a_k ) = T_(m+1)`. 所以 `T_n le U_n`, 由基本不等式的取等条件知, 这里等号成立当且仅当 `a_1 = a_2 = cdots = a_n.`

    (II) 证明 `S_n le T_n`. 由 (I) 知 `1/n ( sum_(k=1)^n 1/a_k ) ge root(n)( prod_(k=1)^n 1/a_k )`, 即 `1/S_n ge 1/T_n`. 所以 `S_n le T_n`. 这里等号成立的条件同 (I).

    (III) 证明 `U_n le V_n`. 由柯西不等式有

    ` ( sum_(k=1)^n 1 * a_k )^2 le ( sum_(k=1)^n 1^2 ) ( sum_(k=1)^n a_k^2 )`,
    即 `U_n le V_n`, 这里等号成立当且仅当 `a_1/1 = a_2/1 = cdots = a_n/1`.

    联系 (I), (II), (III), 对于任意自然数 `n ge 2`, (A1-3-1) 都成立, 且三个等号成立的充要条件均为 `a_1 = a_2 = cdots = a_n`. 证毕.

    A1.4 绝对值不等式

    关于绝对值, 我们已经知道对任意实数 `a` 和非负实数 `m,` 有

    `|a| le m <=> -m le a le m`.
    又设 `b` 是一个实数, 显然有 `|a| le |a|, |b| le |b|`. 根据上式, 这等价于
    `-|a| le a le |a|, -|b| le b le |b|`.
    两式相加,
    `-(|a|+|b|) le a+b le |a|+|b|`,
    这等价于
    `|a+b| le |a|+|b|`. `(A1-4-1)`
    由上式又得
    `|a+b|-|b| le |a|`,
    取 `c=a+b, d=-b,` 上式即
    `|c|-|d| le |c+d|.` `(A1-4-2)`
    对称地, 也有
    `|d|-|c| le |c+d|.` `(A1-4-3)`
    易知 (A1-4-2), (A1-4-3) 中的 `c` 和 `d` 可以取到任意实数. 综合 (A1-4-1) 到 (A1-4-3) 就得到
    `| |a|-|b| | le |a +- b| le |a|+|b|`,
    其中第一个等号成立当且仅当 `ab le 0`; 第二个等号成立当且仅当 `ab ge 0`.

    A1.5 拆项法举例

    例1

    ` 1/n - 1/(n+1)` `= 1/(n(n+1))` `lt 1/(n^2)` `lt 1/(n(n-1))` `= 1/(n-1) - 1/n`.

    例2

    ` 2(sqrt(n+1) - sqrt n)` `= 2/(sqrt(n+1) + sqrt n}` `lt 1/sqrt n` `lt 2/(sqrt n + sqrt(n-1))` `= 2 (sqrt n - sqrt(n-1))`.

    A1.6 杂例

    例1 证明: `log_(n+1) (n+2) lt log_(n) (n+1)`, `n = 2, 3, cdots`.

    因为

    ` sqrt(ln n * ln(n+2))` `lt (ln n + ln(n+2))/2` `= ln sqrt(n(n+2))` `= ln sqrt((n+1)^2-1)` `lt ln(n+1)`
    所以 `(ln(n+2))/(ln(n+1)) lt (ln(n+1))/(ln n)`, 即 `log_(n+1) (n+2) lt log_n (n+1)`, 证毕.

    例2 设 `a, b gt 0`. 证明: 数列 `x_n = root(n)((a^n+b^n)/2)` 单调递增.

    只需证对任意正整数 `n`,

    ` root(n)((a^n+b^n)/2) le root(n+1)(( a^(n+1) + b^(n+1) )/2)`.
    即证
    `(a^n+b^n)^(n+1) le 2( a^(n+1)+b^(n+1) )^n`.
    当 `n=1` 时, 上述不等式即 `(a+b)^2 le 2(a^2+b^2)`, 显然成立. 假设当 `n=k` 时不等式成立, 即 `(a^k+b^k)^(k+1) / ( a^(k+1)+b^(k+1) )^k le 2`. 则 `n=k+1` 时, 断言
    ` ( a^(k+1)+b^(k+1) )^(k+2) / ( a^(k+2)+b^(k+2) )^(k+1) le (a^k+b^k)^(k+1) / ( a^(k+1)+b^(k+1) )^k` `(A1-6-2)`
    成立, 这只要证
    ` ( a^(k+1)+b^(k+1) )^(2(k+1)) le [ (a^k+b^k) ( a^(k+2) + b^(k+2) ) ] ^ (k+1)`,
    即证
    ` ( a^(k+1)+b^(k+1) )^2 le (a^k+b^k)( a^(k+2)+b^(k+2) )`,
    即证
    `2(ab)^(k+1) le (ab)^k (a^2+b^2)`.
    即证 `2ab le a^2+b^2`. 这显然成立. 所以 (A1-6-2) 成立, 从而 `n=k+1` 时不等式成立. 再由数学归纳法知原命题成立.

杂例

  1. `x^4 - 13x^2 + 36`;
  2. `(x^2 + x - 5)(x^2 + x - 7) + 1`;
  3. 证明: `2^1984 + 1` 不是素数;
  4. `x^4 - 4x + 3`;
  5. `x^4 + x^2 y^2 + y^4`;
  6. `x^4 + 2^6`;
  7. `-a^4 -b^4 -c^4 +2b^2 c^2 + 2c^2 a^2 + 2a^2 b^2`;
  8. `x^2 - y^2 + 2x + 6y - 8`;
  9. `4x^2 - 4xy + y^2 - 6x + 3y - 10`;
  10. `a^2 b + ab^2 + a + b - ab - 1`;
  11. `abc + a^2 b - ab^2 - a + b - c`.
  12. `(a + b + c + 1)(a + 1) + bc`;
  13. `(x+y)(y+z)(z+x) + xyz`;
  14. `x^3(a+1) - xy(x-y)(a-b) + y^3(b+1)`;
  15. `x(y^2 - z^2) + y(z^2 - x^2) + z(x^2 - y^2)`;
  16. `a^5 + b^5 - a^3 b^2 - a^2 b^3`;
  17. `x^3 + y^3 + z^3 - 3xyz`;
  1. 将 `x^2` 视为整体, 进行十字相乘. 原式等于 `{: (, (x^2-4)(x^2-9) ),( =, (x+2)(x-2)(x+3)(x-3). ):}`
  2. 同上, 将 `x^2 + x` 视为整体. 原式等于 `{: (, (x^2 + x)^2 -12 (x^2 + x) + 36 ),( =, (x^2 + x - 6)^2 ),( =, [(x-2)(x+3)]^2. ):}`
  3. 实际上是进行因数分解. 当 n 为偶数时, 将公式 ` x^(n+1) - y^(n+1) = (x-y)(x^n + x^(n-1)y + cdots + xy^(n-1) + y^n)` 中的 y 换为 -y 得 ` x^(n+1) + y^(n+1) = (x+y)(x^n - x^(n-1)y + cdots - xy^(n-1) + y^n)`. 于是 `{: (, 2^1984 + 1 ),( =, (2^64)^31 + 1 ),( =, (2^64+1)(cdots). ):}` 显然 `1 lt 2^64+1 lt 2^1984`, 故 `2^1984+1` 不是素数.
  4. 在三项式中, 中项系数的绝对值是另外两个系数的绝对值之和时, 往往可以分解. 原式等于 `{: (, x^4 - x -3x + 3 ),( =, x(x^3-1) -3(x-1) ),( =, (x-1)(x^3 + x^2 + x - 3). ):}`
  5. 有时为凑平方, 我们对中项作改动 (此时往往还需平方差公式). 原式等于 `{: (, x^4 + 2x^2 y^2 + y^4 - x^2 y^2 ),( =, (x^2 + y^2)^2 - (xy)^2 ),( =, (x^2 + xy + y^2)(x^2 - xy + y^2). ):}`
  6. 同上, 原式等于 `{: (, (x^2 + 2^3)^2 - 2*2^3 x^2 ),( =, (x^2 + 4x + 8)(x^2 - 4x + 8). ):}`
  7. 同上, 原式等于 `{: (, 4a^2 b^2 - (a^4+b^4+c^4 - 2b^2 c^2 - 2c^2 a^2 + 2a^2 b^2) ),( =, (2ab)^2 - (a^2 + b^2 - c^2)^2 ),( =, (a^2 + 2ab + b^2 - c^2)(-a^2 + 2ab -b^2 + c^2) ),( =, (a+b+c)(a+b-c)(c+a-b)(c-a+b). ):}` 此式见于三角形面积的海伦公式.
  8. 因为没有形如 cxy 的交叉项, 直接分别对 x, y 配方. 原式等于 `{: (, (x^2 + 2x + 1) - (y^2 - 6y + 9) ),( =, (x+1)^2 - (y-3)^2 ),( =, (x+y-2)(x-y+4). ):}`
  9. 使用 "主元法". 将整个式子视为 x 的多项式, 再将其常数部分视为 y 的多项式. 原式等于 `4x^2 + (-4y-6)x + (y-2)(y+5)`. 为了进行十字相乘, 设上式等于 `(mx + y - 2)(nx + y + 5)`, 于是 `mn = 4`,
    `(m + n)y + 5m - 2n = -4y-6`.     所以 m = n = -2. 于是原式等于 `(-2x + y - 2)(-2x + y + 5)`.
  10. 方法同上. 原式等于 `{: (, ba^2 + (b^2 - b + 1)a + b - 1 ),( =, (a + b - 1)(ba + 1). ):}`
  11. 原式等于 `{: (, ba^2 + (bc - b^2 - 1)a + b - c ),( =, (ba - 1)(a - b + c). ):}`
  12. 原式等于 `{: (, a^2 + (b+c+2)a + bc + b + c + 1 ),( =, a^2 + (b+c+2)a + (b+1)(c+1) ),( =, (a + b + 1)(a + c + 1). ):}`
  13. 即使是看似对称的式子, 主元法依然可以胜任. 原式等于 `{: (, (y+z)x^2 + ((y+z)^2 + yz)x + yz(y+z) ),( =, ((y+z)x + yz)(x + y + z) ),( =, (yz + zx + xy)(x + y + z). ):}`
  14. 在分组分解时, 除主元法, 另一有力手段就是先提 "稀有字母", 即那些出现频率相对较低, 而次数只有一次的字母. 这里我们提出 a, b. 原式等于 `{: (, x^3 a + x^3 - (x^y - xy^2)(a-b) + y^3 b + y^3 ),( =, ax(x^2-xy+y^2) + by(y^2+x^2-xy) + (x+y)(x^2-xy+y^2) ),( =, (ax + by + x + y)(x^2 - xy + y^2). ):}`
  15. 当然可以用主元法. 不过这里用稍特别的方法 (Vandermonde 行列式) 来处理. 原式等于 `|(1,1,1), (x,y,z), (x^2, y^2, z^2)| = (y-x)(z-x)(z-y)`.
  16. 注意到 `(a^3 b^2 + a^2 b^3) = a^2 b^2(a+b)`. `a+b` 是不是 `a^5 + b^5` 的因子呢? 我们用整式除法来验证. 视 a + b 为 a 的一次式, 列表如下: `{: (,1,b), (1,1,b), (-b, -b, -b^2), (b^2, b^2, b^3), (-b^3, -b^3, -b^4), (b^4, b^4, b^5) :}` 恰好整除. 所以原式等于 `(a+b)(1 - b + b^2 - b^3 + b^4 - a^2 b^2)`.
  17. 注意到所给式子中的每一项都在 `(x+y+z)^3` 的展开式中. 猜测 `x + y + z` 是一个因子, 作整式除法: `{: (, 1, y+z), (1, 1, y+z), (-(y+z), -(y+z), -(y+z)^2), (y^2 + z^2 - yz, y^2 + z^2 - yz, y^3 + z^3) :}` 恰好整除. 所以原式等于 `{: (, (x + y + z)(x^2 -(y+z)x + y^2 + z^2 - yz) ),( =, (x + y + z)(x^2 + y^2 + z^2 -yz - zx - xy). ):}`

G4.0 直线的方程

G4.1 直线及其垂线

G4.1.1 点到直线的距离

已知直线 `l: Ax + By + C = 0` 和平面上任意一点 `P_0(x_0, y_0)`, 求 `P_0` 到直线 `l` 的距离.

过 `P_0` 作 `l` 的垂线 `m: B(x-x_0) - A(y-y_0) = 0` 交 `l` 于 `P_1(x_1, y_1)`. 因为 `P_1` 在直线 `m` 上, 所以

`B(x_0-x_1) = A(y_0-y_1)`. `(G4-1-1)`
因为 `P_1` 也在直线 `l` 上, 所以
` Ax_0 + By_0 + C = Ax_0 + By_0 + C - (Ax_1 + By_1 + C)` `= A(x_0-x_1) + B(y_0-y_1)`. `(G4-1-2)`
利用 (G4-1-1) 和 (G4-1-2) 就有
` {:( ,(A^2+B^2) |P_0 P_1|^2 = (A^2 + B^2) [ (x_0-x_1)^2 + (y_0-y_1)^2 ] ),( = ,A^2(x_0-x_1)^2 + B^2(y_0-y_1)^2 + A^2(y_0-y_1)^2 + B^2(x_0-x_1)^2 ),( = ,A^2(x_0-x_1)^2 + B^2(y_0-y_1)^2 + 2AB(x_0-x_1)(y_0-y_1) ),( = ,(Ax_0 + By_0 + C)^2. ):}`
所以
`|P_0 P_1| = |Ax_0 + By_0 + C|/sqrt(A^2 + B^2)`.

G4.1.3 点关于直线的对称点

求 `P_0` 关于直线 `l` 的对称点 `P_2(x_2, y_2)`.

由已知, `P_0 P_2 _|_ l`, 即

`B(x_0-x_2) - A(y_0-y_2) = 0`. `(G4-1-3)`
另一方面, `P_0` 和 `P_2` 的中点 `P_1 ((x_0 + x_2)/2, (y_0 + y_2)/2)` 在直线 `l` 上. 即
`A(x_0+x_2) + B(x_0+x_2) + 2C = 0`. `(G4-1-4)`
由 (G4-1-3) 和 (G4-1-4) 解得
`{ ( x_2 = ((B^2 - A^2)x_0 - 2ABy_0 - 2AC)/(A^2+B^2); ), ( y_2 = ((A^2 - B^2)y_0 - 2ABx_0 - 2BC)/(A^2+B^2). ) :}`
整理得
` ((x_2), (y_2)) = ((x_0), (y_0)) - (2(Ax_0+By_0+C)) / (A^2 + B^2) ((A), (B))`.
上式中, `1/sqrt(A^2 + B^2) ((A), (B))` 相当于直线 `l` 的单位法向量, 而 `(Ax_0 + By_0 + C)/sqrt(A^2 + B^2)` 的绝对值等于 `P_0` 到直线 `l` 的距离.

G4.1.3 点在直线上的射影

由中点的坐标关系知, `P_1` 的坐标为:

` ((x_1), (y_1)) = ((x_0), (y_0)) - ((Ax_0+By_0+C)) / (A^2 + B^2) ((A), (B))`.

1.1 常取 `A^2+B^2 = 1`, 且 `P_0` 为原点. 此时上面三个公式有简单的形式:

`|P_0 P_1| = |C|`;
`((x_1), (y_1)) = -C((A), (B))`;
`((x_2), (y_2)) = -2C((A), (B))`.

G4.2 直线束

G4.2.1 求直线束中过给定点的一条直线方程.

已知 A(2,0), B(0,2), C(1,0), OD `_|_` BC, 交 AB 于 D. 求 CD 的方程.

我们已知 C 的坐标, 只要知道 D 的坐标, CD 的方程自然可以写出, 但 D 的坐标需要通过求解 OD, AB 组成的方程组来获知. 其实使用直线束的知识, 就可以简化解题手续, "猜"出要求的方程.

再者, CD 属于过 D 的直线束, 因此可以表为 OD, AB 的线性组合. 后二者的方程是易得的:

`OD: x-2y = 0,`
(因为 O 在 OD 上, 而 OD 的法向量是 (1,-2));
`AB:x+y-2=0.`
将点 C 代入这两个方程:
`x-2y | _((1,0)) = 1`,
`x+y-2 | _((1,0)) = -1`.
所以 CD 的方程是
`(x-2y)/1 = (x+y-2)/-1`.

另一种方法较繁, 仅供参考: 因为 C 在 CD 上, CD 的方程具有如下形式:

`lambda (x-1) + mu y = 0`.
将 OD 与 AB 的方程改写为
`OD: (x-1)-2y=-1,`
`AB: (x-1)+y=1.`
再相加消去常数项:
`CD: 2(x-1)-y=0.`

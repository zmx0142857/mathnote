几何篇4: 二次曲线

G4.1 定义

定义1 平面上到两点距离之和为定值的点形成的轨迹是椭圆.

如图, 点 `P(x, y)` 到点 `F_1(-c, 0)` 和点 `F_2(c, 0)` 的距离分别为 `r_1,` `r_2,` 且 `r_1 + r_2 = 2a quad` `(0 lt c lt a).` 求 `P` 的轨迹方程.

由已知有

`r_1 = sqrt ((x+c)^2 + y^2); quad (G4-1-1)`

`r_2 = sqrt ((x-c)^2 + y^2); quad (G4-1-2)`

`r_1 + r_2 = 2a. quad (G4-1-3)`

由 (G4-1-1), (G4-1-2) 知

`r_1^2 - r_2^2 = 4cx.`

所以

`r_1 - r_2 = (r_1^2 - r_2^2)/(r_1 + r_2) = 2ex. quad (G4-1-4)`

其中 `e = c/a.`

由 (G4-1-1), (G4-1-2),

`r_1^2 + r_2^2 = 2(x^2 + y^2 + c^2), quad (G4-1-5)`

而同时,

`{: ( r_1^2 + r_2^2 ,= 1/2 [(r_1 - r_2)^2 + (r_1 + r_2)^2] ), ( ,= 1/2 (4 e^2 x^2 + 4 a^2) ), ( ,= 2(e^2 x^2 + a^2) ) :}`

所以

`x^2 + y^2 + c^2 = e^2 x^2 + a^2,`

即

`x^2 / a^2 + y^2 / b^2 = 1.`

其中 `b = sqrt (a^2 - c^2).`

注1 由 (G4-1-3), (G4-1-4) 可得

`{ ( r_1 = a+ex; ), ( r_2 = a-ex. ) :}`

注2 式 (G4-1-5) 也可由 “平行四边形四边的平方和, 等于其对角线的平方和” 看出.

注3 设 `p = a/e,` 则点 `P` 到点 `F_2` 与到直线 `x=p` 的距离之比 `r_2/(p-x) = (a-ex)/(a//e - x) = e.` 事实上, 这个性质也作为椭圆的第二种定义:

定义二 平面上到定点的距离与到定直线的距离之比等于定值 `e` 的点形成的轨迹是圆 (当 `e = 0` ) 或椭圆 (当 `0 lt e lt 1` ) 或抛物线 (当 `e = 1` ) 或双曲线 (当 `e > 1` ).

算法篇3 插值

Lagrange 插值公式

令 `x_1, x_2, cdots, x_n` 是 n 个两两不同的数,

`F(x) = prod_(i=1)^n (x - x_i)`.
我们有
` F'(x) = sum_(i=1)^n (F(x))/(x - x_i) = sum_(i=1)^n prod_(1 le j != i le n) (x - x_j)`.
又令
` l_i(x) = (F(x))/((x - x_i)F'(x_i)) = prod_(1 le j != i le n) (x - x_j)/(x_i - x_j) `.
容易验证 `l_i(x_j) = delta_(ij)`, `i, j = 1, 2, cdots, n`. 其中
`delta_(ij) = { (1, if i = j;), (0, if i != j.) :}`
称为 Kroneker 符号函数. 现在, 函数
` L(x) = sum_(i=1)^n l_i(x) y_i`.
满足条件 `L(x_i) = y_i`, `i = 1, 2, cdots, n`. 这个公式叫做 Lagrange 插值公式. `l_i` 称为插值基函数.

n 个基函数之和为 1. 这是因为

`g(x_j) = sum_(i=1)^n l_i(x_j) = 1`, `j = 1, 2, cdots, n.`
且 `g(x)` 的次数不超过 n-1.

常见线性递推式

`x_(n+1) = a x_n + b` ⇒ `x_n = a^n x_0 + b sum_(i=0)^(n-1) a^i`.

`a = 1` 时, 为等差数列 `x_n = x_0 + bn`;

`a != 1` 时, 设 `x_(n+1) + c = a(x_n + c)`, 得 `c = b/(a-1)`. 于是 `x_n = a^n x_0 + b (a^n - 1)/(a - 1)`.

杂例

  1. `x_n = {(1, if n = 1), (1/n (x_(n-1) + 1/(n!)), if n gt 1):}`; 设 `x_n = 1/(n!) y_n`, 则 `y_1 = 1`, 且 `{: (, 1/(n!) y_n ),( =, 1/n (1/((n-1)!) y_(n-1) + 1/(n!)) ),( =, 1/(n!) (y_(n-1) + 1/n). ):}` 从而 n > 1 时, `y_n = y_(n-1) + 1/n`. 所以 `y_n = sum_(k=1)^n 1/k`, `x_n = 1/(n!) sum_(k=1)^n 1/k`.

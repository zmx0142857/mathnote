Nabla 算子

`grad = (del/(del x_1), cdots, del/(del x_m))`

`grad c = 0` `grad * bm c = 0`
`grad (cf) = c grad f` `grad * (c bm F) = c (grad * bm F)`
`grad (f +- g) = grad f +- grad g` `grad * (bm F +- bm G) = grad * bm F +- grad * bm G`
`grad (fg) = g grad f + f grad g` `grad * (f bm F) = grad f * bm F + f (grad * bm F)`
`grad (f/g) = (g grad f - f grad g) / g^2` `grad times bm c = bm 0`
`grad f(g) = (df)/(dg) grad g` `grad times (c bm F) = c grad times bm F`
`grad f(g_1, cdots, g_n) = sum (del f)/(del g_i) grad g_i` `grad times (bm F +- bm G) = grad times bm F +- grad times bm G`
`grad * (grad f) = Delta f` `grad times (f bm F) = grad f times bm F + f(grad times bm F)`
`grad * (grad times bm F) = 0` `grad times (grad f) = bm 0`

Laplace 算子

`Delta = sum del^2/(del x_i^2)`

`Delta c = 0`
`Delta (cf) = c Delta f`
`Delta (f +- g) = Delta f +- Delta g`
`Delta (fg) = f Delta g + g Delta f + 2 grad f * grad g`
`Delta f(g) = grad (df)/(dg) * grad g + (df)/(dg) Delta g`
`Delta f(g_1, cdots, g_n) = sum grad (del f)/(del g_i) * grad g_i + sum (del f)/(del g_i) Delta g_i`

径向函数

设 `r = sqrt(sum_(i=1)^m x_i^2)`, 则 `grad r = 1/r (x_1, cdots, x_m)`
`grad r * grad r = 1`
`grad f(r) = (df)/(dr) grad r`
`Delta r = (m-1)/r`
`Delta f(r) = (d^2 f)/(dr^2) + (m-1)/r (df)/(dr)`
`r Delta f = d^2/(dr^2) (rf) quad (m = 3)`

曲线/曲面积分常用记号

曲线积分

曲线参数微元 (时间) `dt`
坐标长度微元 `dx, dy, dz`
位移矢量微元 `d bm r = (dx, dy, dz)`
切向量 `(d bm r)/(dt) = ((dx)/(dt), (dy)/(dt), (dz)/(dt))`
外法向量 `bm n = (dy/dt, -dx/dt)`
弧长微元 `ds = |(d bm r)/dt| dt`
单位切向量 `bm t = (d bm r)/dt |(d bm r)/(dt)|^-1`
联系 `d bm r = bm t ds`
验证 `d bm r = (d bm r)/dt dt = bm t ds.`

曲面积分

曲面参数微元 `du, dv`
坐标平面上的面积微元 `dydz, dzdx, dxdy`
曲面积分微元 `d bm S = (dydz, dzdx, dxdy)`
沿参数方向的切向量 `bm r_u = ((del x)/(del u), (del y)/(del u), (del z)/(del u))` `bm r_v = ((del x)/(del v), (del y)/(del v), (del z)/(del v))`
法向量 `bm r_u times bm r_v`
曲面面积微元 `d sigma = |bm r_u times bm r_v| dudv`
单位外法向量 `bm n = +- (bm r_u times bm r_v)/|bm r_u times bm r_v|`
联系 `d bm S = bm n d sigma`
验证 `d bm S = ( (del(y,z))/(del(u,v)), (del(z,x))/(del(u,v)), (del(x,y))/(del(u,v)) ) dudv` `= bm r_u times bm r_v dudv = bm n d sigma.`

多元积分公式

Leibniz: `d/dt int_(alpha(t))^(beta(t)) f(x, t) dx` `= int_(alpha(t))^(beta(t)) del/(del t) f(x, t) dx` `+ f(beta(t), t) beta'(t)` `- f(alpha(t), t) alpha'(t).` Green: `∬_D |(del/(del x), del/(del y)), (P, Q)| dxdy` `= oint_(del D) P dx + Q dy` (`del y` 者符号相反)
`∬_D Delta u dxdy` `= oint_(del D) (del u)/(del bm n) ds`
单位外法向 `bm n = (dy/(ds), -dx/(ds))` Gauss: `∭_Omega ((del P)/(del x) + (del Q)/(del y) + (del R)/(del z)) dxdydz` `= ∯_(del Omega) P dydz + Q dzdx + R dxdy`
`∭_Omega grad * bm F dV` `= ∯_(del Omega) bm F * d bm S`
`∭_Omega Delta u dxdydz` `= ∯_(del Omega) (del u)/(del bm n) d sigma`
单位外法向 `bm n` Stokes: `∬_S | (dydz, dzdx, dxdy), (del/(del x), del/(del y), del/(del z)), (P, Q, R) |` `= oint_(del S) P dx + Q dy + R dz`
`∬_S grad times bm F * d bm S` `= oint_(del S) bm F * d bm r`

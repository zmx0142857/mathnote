Nabla 算子

`grad = (del/(del x_1), cdots, del/(del x_m))`

`grad c = 0` `grad * bb"c" = 0`
`grad (cf) = c grad f` `grad * (c bb"F") = c (grad * bb"F")`
`grad (f +- g) = grad f +- grad g` `grad * (bb"F" +- bb"G") = grad * bb"F" +- grad * bb"G"`
`grad (fg) = g grad f + f grad g` `grad * (f bb"F") = grad f * bb"F" + f (grad * bb"F")`
`grad (f/g) = (g grad f - f grad g) / g^2` `grad times bb"c" = bb"0"`
`grad f(g) = (df)/(dg) grad g` `grad times (c bb"F") = c grad times bb"F"`
`grad f(g_1, cdots, g_n) = sum (del f)/(del g_i) grad g_i` `grad times (bb"F" +- bb"G") = grad times bb"F" +- grad times bb"G"`
`grad * (grad f) = Delta f` `grad times (f bb"F") = grad f times bb"F" + f(grad times bb"F")`
`grad * (grad times bb"F") = 0` `grad times (grad f) = bb"0"`

Laplace 算子

`Delta = sum del^2/(del x_i^2)`

`Delta c = 0`
`Delta (cf) = c Delta f`
`Delta (f +- g) = Delta f +- Delta g`
`Delta (fg) = f Delta g + g Delta f + 2 grad f * grad g`
`Delta f(g) = grad (df)/(dg) * grad g + (df)/(dg) Delta g`
`Delta f(g_1, cdots, g_n) = sum grad (del f)/(del g_i) * grad g_i + sum (del f)/(del g_i) Delta g_i`

径向函数

设 `r = sqrt(sum_(i=1)^m x_i^2)`, 则 `grad r = 1/r (x_1, cdots, x_m)`
`grad r * grad r = 1`
`grad f(r) = (df)/(dr) grad r`
`Delta r = (m-1)/r`
`Delta f(r) = (d^2 f)/(dr^2) + (m-1)/r (df)/(dr)`
`r Delta f = d^2/(dr^2) (rf) quad (m = 3)`

曲线/曲面积分常用记号

曲线参数微元 (时间) `dt` 曲面参数微元 `du, dv`
坐标微元 `dx, dy, dz` 坐标平面上的面积微元 `dydz, dzdx, dxdy`
位移矢量微元 `d bb"r" = (dx, dy, dz)` ??? `d bb"S" = (dydz, dzdx, dxdy)`
切向量 `(d bb"r")/(dt) = ((dx)/(dt), (dy)/(dt), (dz)/(dt))` 沿参数方向的切向量 `bb"r"_u = ((del x)/(del u), (del y)/(del u), (del z)/(del u))` `bb"r"_v = ((del x)/(del v), (del y)/(del v), (del z)/(del v))`
法向量 `bb"n" = (dy/dt, -dx/dt)` 法向量 `bb"r"_u times bb"r"_v`
弧长微元 `ds = |(d bb"r")/dt| dt` 曲面面积微元 `d sigma = |bb"r"_u times bb"r"_v| dudv`
单位切向量 `bb"t" = (d bb"r")/dt |(d bb"r")/(dt)|^-1` 单位外法向量 `bb"n" = +- (bb"r"_u times bb"r"_v)/|bb"r"_u times bb"r"_v|`
联系 `d bb"r" = bb"t" ds` 联系 `d bb"S" = bb"n" d sigma`
验证 `d bb"r" = (d bb"r")/dt dt = bb"t" ds.` 验证 `d bb"S" = ( (del(y,z))/(del(u,v)), (del(z,x))/(del(u,v)), (del(x,y))/(del(u,v)) ) dudv` `= bb"r"_u times bb"r"_v dudv = bb"n" d sigma.`

多元积分公式

Leibniz: `d/dt int_(alpha(t))^(beta(t)) f(x, t) dx` `= int_(alpha(t))^(beta(t)) del/(del t) f(x, t) dx` `+ f(beta(t), t) beta'(t)` `- f(alpha(t), t) alpha'(t).` Green: `∬_D |(del/(del x), del/(del y)), (P, Q)| dxdy` `= oint_(del D) P dx + Q dy` (`del y` 者符号相反)
`∬_D Delta u dxdy` `= oint_(del D) (del u)/(del bb"n") ds`
单位外法向 `bb"n" = (dy/(ds), -dx/(ds))` Gauss: `∭_Omega ((del P)/(del x) + (del Q)/(del y) + (del R)/(del z)) dxdydz` `= ∯_(del Omega) P dydz + Q dzdx + R dxdy`
`∭_Omega grad * bb"F" dV` `= ∯_(del Omega) bb"F" * d bb"S"`
`∭_Omega Delta u dxdydz` `= ∯_(del Omega) (del u)/(del bb"n") d sigma`
单位外法向 `bb"n"` Stokes: `∬_S | (dydz, dzdx, dxdy), (del/(del x), del/(del y), del/(del z)), (P, Q, R) |` `= oint_(del S) P dx + Q dy + R dz`
`∬_S grad times bb"F" * d bb"S"` `= oint_(del S) bb"F" * d bb"r"`

杂例

例1 证明: `|P(AB) - P(A)P(B)| le 1/4`, 并讨论等号成立的条件.

先证: 对任意事件 A, B, 有 `min {P(A)P(B), P(bar A)P(bar B)} le 1/4`. (S2-1) 假设 `P(bar A)P(bar B) ge 1/4`, 有 `1/4 le P(bar A)P(bar B) le 1/4 [P(bar A) + P(bar B)]^2`, 即 `P(bar A) + P(bar B) ge 1`. 则 ` P(A)P(B) le ((P(A) + P(B))/2)^2 = 1/4 {2 - [P(bar A) + P(bar B)]}^2 le 1/4`. 所以 (S2-1) 成立. 这里取等号的条件是 `P(A)P(B) = P(bar A)P(bar B) = 1/4`, 解得 `P(A) = P(B) = 1/2`.

再来讨论题目所要证的不等式. 由于 `0 le P(AB) le P(A)`,
`0 le P(AB) le P(B)`,
`0 le P(A uu B) = P(A) + P(B) - P(AB) le 1`. 所以 `max {0, P(A) + P(B) - 1} le P(AB) le min {P(A), P(B)}`. (S2-2) 不等号两边同减 `P(A)P(B)`, 得 ` max {-P(A)P(B), -P(bar A)P(bar B)} le P(AB) - P(A)P(B)` `le min {P(A)P(bar B), P(bar A)P(B)}`. 再应用 (S2-1) 就得到要证的不等式. 因为 (S2-2) 取等号的条件是 `P(AB) = P(A)` 或 `P(AB) = P(B)` 或 `P(AB) = 0` 或 `P(AB) = P(A) + P(B) - 1` 所以原不等式取等号的条件就是 (S2-1), (S2-2) 同时取得等号, 即 `P(A) = P(B) = P(AB) = 1/2` 或 `P(A) = P(B) = 1/2, P(AB) = 0`. □

分析篇10: 数项级数

D'Alembert's Test (达朗贝尔检比法)

若正项级数的一般项 `u_n` 满足

`lim_(n to oo) (u_(n+1)) / (u_n) = l`.
则

`0 le l lt 1` 时, 级数收敛;

`1 lt l le oo` 时, 级数发散.

`0 le l lt 1` 时, 取实数 r 使 `l lt r lt 1`. 则由 `lim_(n to oo) (u_(n+1)) / (u_n) = l` 知 对 `epsilon = r-l gt 0` 存在正整数 N, 使对所有 n > N 成立 `u_(n+1) lt (l+epsilon) u_n = r u_n`.

于是 `u_n lt r u_(n-1) lt cdots lt r^(n-N) u_N`. 所以由级数 `sum_(n=N)^(oo) u_N r^(n-N)` 收敛知 `sum_(n=N)^(oo) u_n` 也收敛, 从而 `sum_(n=1)^(oo) u_n` 收敛.

`1 lt l le oo` 时, 取实数 r 使 `1 lt r lt l` (`l = oo` 时, 任取 r > 1), 则对 `epsilon = l-r gt 0` 存在正整数 N, 使对所有 n > N 成立 `u_(n+1) gt (l-epsilon) u_n = r u_n`.

于是 `u_n gt r u_(n-1) gt cdots gt r^(n-N) u_N`. 但级数 `sum_(n=N)^(oo) u_N r^(n-N)` 发散, 所以 `sum_(n=N)^(oo) u_n`, 从而 `sum_(n=1)^(oo) u_n` 发散.

这一定理可以推广到复数级数甚至矩阵级数. 以后者为例, 可以叙述为: 若矩阵级数的一般项 `U_(n+1) = V_n U_n`, 其中

`lim_(n to oo) ||V_n|| = l`.
则 `0 le l lt 1` 时, 级数 `sum_(n=1)^(oo) norm(U_n)` 收敛, 从而 `sum_(n=1)^(oo) U_n` 收敛. 我们可以用下面的不等式证明之:
`||AB|| le ||A|| ||B||`.

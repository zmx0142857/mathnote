几何篇0: 向量

G0.6 证明: 平行四边形四边长的平方和 = 对角线长的平方和.

证 用向量轻松证明:

`{: ( ,|bb"a" + bb"b"|^2 + |bb"a" - bb"b"|^2 ), ( = ,(bb"a" + bb"b")*(bb"a" + bb"b") + (bb"a" - bb"b")*(bb"a" - bb"b") ), ( = ,2|bb"a"|^2 + 2|bb"b"|^2. ) :}`

推论G0.6.1 设三角形三边长为 a, b, c, 则边 a 上的中线 `m_a` 满足:

`4 m_a^2 = 2(b^2 + c^2) - a^2.`

推论G0.6.2 任意平面四边形四边长的平方和 = 对角线的平方和 + 4 `xx` 对角线中点连线长的平方.

证 设四边形 ABCD 的对角线 AC 的中点为 M, 对角线 BD 的中点为 N. 在 `Delta NAC, Delta ABD, Delta CBD` 中分别使用推论G0.6.1, 有

`4 MN^2 = 2 (AN^2 + CN^2) - AC^2,`

`4 AN^2 = 2 (AB^2 + AD^2) - BD^2,`

`4 CN^2 = 2 (CB^2 + CD^2) - BD^2.`

将后两个式子代入第一个式子右边, 整理就得到

`AB^2 + BC^2 + CD^2 + DA^2 = AC^2 + BD^2 + 4MN^2.`

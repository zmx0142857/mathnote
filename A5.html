分析篇5: 关于 `sum_(k=1)^n k^m` 的一些结论

注A5.1 本文实际上讨论伯努利数, 文中的所谓 `a_n`, 就是 (第二类) 伯努利数 `B_n^+`. 请参考 维基百科.

可以用如下方法求 `sum_(k=1)^n k^2` 的多项式表达: 因为

`(k+1)^3 = k^3 + 3k^2 + 3k + 1`,
移项
`3k^2 = (k+1)^3 - k^3 - 3k - 1`,
两边求和
` {:( 3 sum_(k=1)^n k^2 ,= sum_(k=1)^n [(k+1)^3 - k^3] - 3 sum_(k=1)^n k - sum_(k=1)^n 1 ),( ,= (n+1)^3 - 1 - (3n(n+1))/2 - n ),( ,= n^3 + 3/2 n^2 + 1/2 n. ):}`
所以
`sum_(k=1)^n k^2 = 1/3 n^3 + 1/2 n^2 + 1/6 n`.

这种方法对于求 `sum_(k=1)^n k^m` (`m` 为自然数) 亦不失一般性. 然而从上面的计算已经看到, 为了得到 `sum_(k=1)^n k^2` 的表达式, 分别用了 `sum_(k=1)^n k` 与 `sum_(k=1)^n 1` 的结果. 以这种方法计算 `sum_(k=1)^n k^m`, 工作量是巨大的: 要计算 `sum_(k=1)^n 1`, `sum_(k=1)^n k`, `sum_(k=1)^n k^2`, `cdots`, `sum_(k=1)^n k^(m-1)`, 共 `m` 个结果!

为了方便地得到 `sum_(k=1)^n k^m` 的多项式表达, 有必要进一步探讨它的性质. 下面我们记 `sum_(k=1)^n k^m = S_m(n)`, 或简记为 `S_m`.

性质一 对任意自然数 `m`, `S_m(n)` 是关于 `n` 的 `m+1` 次多项式, 其常数项为 `0`.

当 `m=0` 时, 易知 `S_0(n) = n`, 满足性质一. 假设性质一对于自然数 `0` 到 `m-1` 都成立, 考察自然数 `m` 的情形:

由二项式定理

` (k+1)^(m+1) = k^(m+1) + (m+1)k^m + sum_(i=0)^(m-1) ((m+1), (i)) k^i`,
移项
` (m+1)k^m = (k+1)^(m+1) - k^(m+1) - sum_(i=0)^(m-1) ((m+1),(i)) k^i`,
两边求和
` (m+1) sum_(k=1)^n k^m = (n+1)^(m+1) - 1 - sum_(i=0)^(m-1) ((m+1),(i)) sum_(k=1)^n k^i`. `(A5-1)`
由归纳假设, 上式等号右端的最后一项是 (关于 `n` 的) 常数项为 `0` 的 `m+1` 次多项式; 而 `(n+1)^(m+1) - 1` 显然也是如此. 所以 `S_m(n)` 一定是关于 `n` 的常数项为 `0` 的 `m+1` 次多项式. □

性质二 `S_m(n)` 各次项系数之和等于 `1`.

取 n=1 立得. □

性质三 设 `a_m` 是 `S_m(n)` 的一次项系数, 则
`a_m = 1 - 1/(m+1) sum_(i=0)^(m-1) ((m+1),(i)) a_i`. `(A5-2)`

在式 (A5-1) 两端对 `n` 求导, 再同除以 `m+1`, 得

` text(d)/(text(d)n) S_m(n) = (n+1)^m - 1/(m+1) sum_(i=0)^(m-1) ((m+1),(i)) text(d)/(text(d)n) S_i`. `(A5-3)`
由上式两端的常数项相等, 即知性质三成立. □

性质四 `text(d)/(text(d)n) S_m - a_m = mS_(m-1)`.

`m=1` 时, ` text(d)/(text(d)n) S_1 - a_1 = text(d)/(text(d)n) ( n^2/2 + n/2 ) - 1/2 = n = 1 * S_0`, 性质四成立. 假设性质四对于正整数 `1, 2, cdots, m-1` 已成立, 考察正整数 `m` 的情形:

由 (A5-2) 与 (A5-3) 有

` {:( text(d)/(text(d)n) S_m - a_m ,= (n+1)^m - 1 - 1/(m+1) sum_(i=0)^(m-1) ((m+1),(i)) ( text(d)/(text(d)n) S_i - a_i ) ),( ,= (n+1)^m - 1 - 1/(m+1) sum_(i=0)^(m-1) ((m+1),(i)) i S_(i-1) ),( ,= (n+1)^m - 1 - sum_(i=0)^(m-2) (i+1)/(m+1) ((m+1),(i)) S_i ),( ,= (n+1)^m - 1 - sum_(i=0)^(m-2) ((m),(i)) S_i. ):}`
再由 (A5 - 1),
`m S_(m-1) = (n+1)^m - 1 - sum_(i=0)^(m-2) ((m),(i)) S_i`.
所以 `text(d)/(text(d)n) S_m - a_m = m S_(m-1)`. 由第二数学归纳法知, 性质四对于任意正整数成立. □

注A5.2 借助性质三、四, 我们将 `S_m` 的系数排列如下:

`{: ( m, text(Coefficient), S_m(n) ), ( 0, 1, n ), ( 1, 1/2 \ 1/2, 1/2 n^2 + 1/2 n ), ( 2, 1/3 \ 1/2 \ 1/6, 1/3 n^3 + 1/2 n^2 + 1/6 n ), ( 3, 1/4 \ 1/2 \ 1/4 \ 0, 1/4 n^4 + 1/2 n^3 + 1/4 n^2 ), ( 4, 1/5 \ 1/2 \ 1/3 \ 0 \ -1/30, 1/5 n^5 + 1/2 n^4 + 1/3 n^3 - 1/30 n ) :}`
在系数三角形中, `m` 行 `k` 列 (`m, k` 从 `0` 开始计数) 的元素可以由 `m-1` 行 `k` 列的元素乘以 `m/(m+1-k)` 得到. 于是该元素等于
`a_k * (m!)/((m+1-k)!k!) = ((m),(k)) a_k/(m+1-k)`.
所以
` S_m(n) = sum_(k=1)^n k^m = sum_(k=0)^m ((m),(k)) a_k/(m+1-k) n^(m+1-k)`.
或
`S_m(n) = 1/(m+1) sum_(k=1)^(m+1) ((m+1),(k)) a_(m+1-k) n^k`.
( 对比 `1/(m+1) (1+n)^(m+1) = 1/(m+1) sum_(k=0)^(m+1) ((m+1),(k)) n^k` )

性质五 `m ge 1` 时, 函数 `T(n) = S_m(n) - n^m/2` 的奇偶性与 `m` 相反, 即
  • `m = 1, 3, 5, cdots` 时, `T(n)` 是偶函数;
  • `m = 2, 4, 6, cdots` 时, `T(n)` 是奇函数.

对函数 `T(n)` 求 `m` 阶导数. 由性质四

` {:( S_m^'(n) ,= m S_(m-1)(n) + a_m; ),( S_m^('')(n) ,= m S'_(m-1)(n) = m(m-1) S_(m-2)(n) + m a_(m-1); ),( cdots, ),( S_m^((m)) (n) ,= m! (S_0(n) + a_1) = m! (n + 1/2). ):}`
于是 `T^((m))(n) = (S_m(n) - n^m/2)^((m)) = m!n`, 这显然是奇函数.

我们知道, 奇函数的原函数是偶函数, 偶函数的原函数中图象过原点的一个是奇函数. 由性质一, `S_k(n) quad (k = 1, 2, cdots, m)` 的图象过原点, 所以知 `T(n)` 的奇偶性与 `m` 相反. □

性质六 由性质五立即推得 `a_3 = a_5 = a_7 = cdots = 0`. (它们是偶函数的一次项系数).

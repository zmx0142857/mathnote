方程篇3: 微分方程初等解法

杂例

`dy/dx = (2x^3 + 3xy^2 + x) / (3x^2 y + 2y^3 - y)`.

原方程化为

`(y dy)/(x dx) = (2x^2 + 3y^2 + 1) / (3x^2 + 2y^2 - 1)`.
令 `X = x^2 - 1`, `Y = y^2 + 1`, 则
`(dY)/(dX) = (2X + 3Y)/(3X + 2Y)`.
又令 `u = Y/X`, 则 `Y = uX`, `(dY)/(dX) = u + X (du)/(dX)`. 方程化为
`(2 + 3u)/(3 + 2u) = u + X(du)/(dX)`.
分离变量得
`2 (dX)/X = (3 + 2u)/(1-u^2) du`.
两边积分,
`2 ln |X| + c_1 = 3/2 ln |(1+u)/(1-u)| - ln |1-u^2|`.
即
`c_2 X^2 = (1+u)^(1/2) (1-u)^(-2/5)`.
两边平方并代回原变量,
`c (x^2-1)^4 = (1 + (y^2+1)/(x^2-1))(1-(y^2+1)/(x^2-1))^(-5)`.
即
`c(x^2 - y^2 - 2)^5 = (x^2 + y^2)`.

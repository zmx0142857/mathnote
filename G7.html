几何篇7: 二次曲线

定义

平面上到两点距离之和为定值的点形成的轨迹是椭圆.

如图, 点 `P(x, y)` 到点 `F_1(-c, 0)` 和点 `F_2(c, 0)` 的距离分别为 `r_1`, `r_2`, 且 `r_1 + r_2 = 2a` `(0 lt c lt a)`. 求 `P` 的轨迹方程.

由已知有

`r_1 = sqrt ((x+c)^2 + y^2)`; (G7-1)
`r_2 = sqrt ((x-c)^2 + y^2)`; (G7-2)
`r_1 + r_2 = 2a`. (G7-3)
由 (G7-1), (G7-2) 知
`r_1^2 - r_2^2 = 4cx`.
所以
`r_1 - r_2 = (r_1^2 - r_2^2)/(r_1 + r_2) = 2ex`. (G7-4)
其中 `e = c/a`.

由 (G7-1), (G7-2),

`r_1^2 + r_2^2 = 2(x^2 + y^2 + c^2)`, (G7-5)
而同时,
` {:( r_1^2 + r_2^2 , = 1/2 [(r_1 - r_2)^2 + (r_1 + r_2)^2] ),(, = 1/2 (4 e^2 x^2 + 4 a^2) ),(, = 2(e^2 x^2 + a^2) ):}`
所以
`x^2 + y^2 + c^2 = e^2 x^2 + a^2`,
即
`x^2 / a^2 + y^2 / b^2 = 1`.
其中 `b = sqrt (a^2 - c^2)`.

由 (G7-3), (G7-4) 可得

`r_1 = a + ex`;

`r_2 = a - ex`.

式 (G7-5) 也可由 “平行四边形四边的平方和, 等于其对角线的平方和” 看出.

设 `p = a/e`, 则点 `P` 到点 `F_2` 与到直线 `x = p` 的距离之比 `r_2/(p-x) = (a-ex)/(a//e - x) = e`. 事实上, 这个性质也作为椭圆的第二种定义:

平面上到定点的距离与到定直线的距离之比等于定值 `e` 的点形成的轨迹 是圆 (当 `e = 0` ) 或椭圆 (当 `0 lt e lt 1` ) 或抛物线 (当 `e = 1` ) 或双曲线 (当 `e gt 1` ).

圆锥曲线的切线

椭圆 `x^2/a^2 + y^2/b^2 = 1` (a, b > 0) 过点 `(x_0, y_0)` 的切 线方程为

`(x_0 x)/a^2 + (y_0 y)/b^2 = 1`.

在椭圆方程的两端对 x 求导,

`(2x)/a^2 + (2y)/b^2 y^' = 0`.
上式在 `(x_0, y_0)` 也成立, 因此切线的方程为

`y - y_0 = y^'|_((x_0, y_0)) (x - x_0)`,

`y - y_0 = -(x_0 b^2)/(y_0 a^2) (x - x_0)`,

`x_0 b^2 (x - x_0) + y_0 a^2 (y - y_0) = 0`,

`b^2 x_0 x + a^2 y_0 y = b^2 x_0^2 + a^2 y_0^2`,

因为
`b^2 x_0^2 + a^2 y_0^2 = a^2 b^2`,
所以切线方程为
`(x_0 x)/a^2 + (y_0 y)/b^2 = 1`.
易知斜率不存在时, `y_0 = 0`, 上式也是椭圆的切线方程. □

如利用 Δ = 0, 则计算繁琐: 联立

` {( A(x - x_0) + B(y - y_0) = 0; ),( x^2/a^2 + y^2/b^2 = 1. ):}`
令 Δ = 0, 得
`A^2 a^2 + B^2 b^2 = (A x_0 + B y_0)^2`.
B ≠ 0 时, 记 `m = A/B`, 解上述二次方程 (因为
`b^2 x_0^2 + a^2 y_0^2 = a^2 b^2`,
这个方程有两个相等实根), 得
` {:( m , = (x_0 y_0)/(a^2 - x_0^2) = x_0/(a^2 y_0) (a^2 y_0^2)/(a^2 - x_0^2) ),(, = x_0/(a^2 y_0) (a^2 b^2 - b^2 x_0^2)/(a^2 - x_0^2) = (x_0 b^2)/(y_0 a^2). ):}`
再作类似处理可以得到切线方程. □

杂例

二元二次方程及其表示的曲线

` ( ( a_(11)^', a_(12)^' ), ( a_(12)^', a_(22)^' ) ) = ( ( cos theta, sin theta ), ( -sin theta, cos theta ) ) ( ( a_(11), a_(12) ), ( a_(12), a_(22) ) ) ( ( cos theta, -sin theta ), ( sin theta, cos theta ) ).`

计算知

`a_(11)^' = a_(11) cos^2 theta + a_(22) sin^2 theta + 2a_(12) sin theta \ cos theta`;
`a_(22)^' = a_(11) sin^2 theta + a_(22) cos^2 theta - 2a_(12) sin theta \ cos theta`;
`a_(12)^' = a_(12) (cos^2 theta - sin^2 theta) - (a_(11) - a_(22)) sin theta \ cos theta`.
所以
`a_(11)^' + a_(22)^' = a_(11) + a_(22)`;
` {:(, a_(11)^' - a_(22)^' = (a_(11) - a_(22)) (cos^2 theta - sin^2 theta) + 4 a_(12) sin theta \ cos theta ),( =, (a_(11) - a_(22)) cos 2theta + 2 a_(12) sin 2 theta. ):}`
由 `a_(12)^' = 0` 有 `cot 2 theta = (a_(11)-a_(22)) / (2a_(12))`, 故
` {:( a_(11)^' , = (a_(11) + a_(22))/2 + (a_(11) - a_(22))/2 cos 2 theta + a_(12) sin 2 theta ),(, = a_(11) - (a_(11) - a_(22))/2 + (a_(11) - a_(22))/2 cos 2 theta + a_(12) sin 2 theta ),(, = a_(11) - (a_(11) - a_(22))/2 (1 - cos 2 theta) + a_(12) sin 2 theta ),(, = a_(11) + a_(12) [sin 2 theta - cot 2 theta(1 - cos 2 theta)] ),(, = a_(11) + a_(12) (sin^2 2 theta - cos 2 theta (1- cos 2 theta)) / (sin 2 theta) ),(, = a_(11) + a_(12) (1-cos 2 theta) / (sin 2 theta) ),(, = a_(11) + a_(12) tan theta. ):}`
` a_(22)^' = a_(11) + a_(22) - a_(11)^' = a_(22) - a_(12) tan theta`.
